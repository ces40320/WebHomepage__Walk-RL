Filename Rules

Videos are named using tokens that describe the main control mode and the evaluation environment.

Token reference

Main Group Condition

  • ind_: Independent
  • syn_: Synergistic

Environment

Training Env

  • _uneven: Randomized Uneven Terrain (±6°)

(This video is representive terrain at 1.2 m/s walking speed)

Evaluating Env

  • _{float.1}mps: {speed} m/s
    • If _frontal: on frontal plane
    • Else: on sagittal plane
  • _downhill: Ramp Descent (+5°)
  • _uphill: Ramp Ascent (-5°)
